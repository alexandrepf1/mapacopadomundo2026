# # FIFA World Cup 2026 Match Map

Interactive map for exploring FIFA World Cup 2026 matches by date, team, stage, venue, host country, and timezone.

## Features

- Interactive match map
- Date and team filters
- Timezone selector
- Venue, stage, and host-country filters
- Match cards with kickoff time, score, venue, and city
- Refresh button for score updates using ESPN data when available
- Link to FIFA Match Centre

## Data note

This is an unofficial fan-made project. Match schedule and venue data are embedded in the page. Live score refresh depends on ESPN’s public scoreboard data and may stop working if ESPN changes or blocks its endpoint.

## Official source

For official match information, use FIFA Match Centre:

https://www.fifa.com/en/match-centre

## Privacy

This page loads third-party resources such as map tiles, Leaflet assets, flag images, and ESPN score data. These services may receive standard browser request information such as IP address and user agent.
